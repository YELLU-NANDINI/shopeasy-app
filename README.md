# 🛍️ ShopEasy - E-commerce Web App

## 📌 Project Overview

ShopEasy is a modern e-commerce web application built using Next.js.
It allows users to browse products, view details, add items to cart, and simulate checkout — all with a smooth UI.

---

## 🚀 Features

* 🛒 Product Listing (API based)
* 🔍 Search Products
* 📄 Product Details Page
* 🛍️ Add to Cart
* ➕➖ Quantity Management
* ❌ Remove Items
* 💰 Total Price Calculation
* 🔐 Login & Signup (Frontend Only)
* 🧾 Checkout with Address
* 🔔 Toast Notifications
* 🧠 State Management using Zustand
* 🧪 Basic Testing using Jest

---

## 🛠️ Tech Stack

* Next.js (App Router)
* React
* TypeScript
* Tailwind CSS
* Zustand (State Management)
* React Hot Toast
* Jest & Testing Library

## ▶️ How to Run

1. Clone the repo
2. Install dependencies

npm install

3. Run the project

npm run dev

4. Open browser:

http://localhost:3000
